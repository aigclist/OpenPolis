# Skills System

Skills are explicit units of capability. They are not hidden prompts or opaque background automations.

## Each skill defines

- A bounded scope
- The object types it can read or draft
- Whether it is advisory or state-changing
- Which actions require human confirmation

## Current seed skills

- Issue summarizer
- Brief drafter
- Approval guard

These skills are intentionally workflow-bound. They do not publish external content and they do not run outreach or persuasion automations.
