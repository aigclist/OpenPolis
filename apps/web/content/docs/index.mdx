# OpenPolis Overview

OpenPolis is an open core for mission-driven organizations that need structured issues, reusable assets, accountable operations, and reviewable AI assistance in the same workspace.

## Product posture

- Task-structured information instead of content silos
- Human review gates before external impact
- Self-hosted deployment as the default assumption
- Auditability treated as a product feature rather than a compliance afterthought

## Technical posture

- Next.js 16 App Router baseline
- `next-intl` messages for application copy
- Tailwind v4 with shared semantic tokens
- `shadcn/ui` and AI Elements as source-controlled UI primitives
- Fumadocs for principles, architecture, and implementation notes
